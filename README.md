# Calculadora de Precificação

Calculadora de precificação para autônomos e micro-empresários brasileiros.

**Stack:** Next.js 14 · TypeScript · Drizzle ORM · libsql (Turso)

## Setup local

```bash
npm install
cp .env.local .env.local
# Preencha DATABASE_URL e DATABASE_AUTH_TOKEN no .env.local
npm run dev
```

## Build

```bash
npm run build
npm start
```
